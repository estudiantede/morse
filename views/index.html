El código morse

Escrito por Iñaki Frutos

¿Qué es el código morse?

Es un sistema de comunicación que tiene representación mediante señales intermitentes que son llamadas usualmente puntos y rayas

Se usaba antes para la comunicación mediante el telegrafo (inventaod por Samuel Morse), pero ahora no se usa tanto como fue en su época, en la cuál los barcos la utilizaban, así como los aviones. Pero sigue siendo utilizado en caso de no poder comunicarse por radio, así que sigue siendo una herramienta útil, en cierto sentido

Cómo funciona el código morse

El código morse utiliza puntos y rayas de forma intermitente. Esto sirve para poder representar a todas las letras del alfabeto inglés, junto con algunos simbolos de puntación
